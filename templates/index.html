Created by Guido van Rossum in 1991.
It is easy to read, write, and learn.
Used in Web Development, Data Science, Machine Learning, Automation, AI, etc.

Open Source → Free to use.
Huge Library Support → Many pre-built modules.
Cross-platform → Works on Windows, Linux, macOS.
Interpreted Language → No need to compile; executes line by line.

✔ Easy to Learn and Read
✔ Supports Object-Oriented and Procedural Programming
✔ Dynamic Typing (no need to declare variable type)
✔ Large Standard Library (like math, os, sys)
✔ Automatic Memory Management (Garbage Collection)
✔ Extensible and Embeddable

Expression and statements


🔹 1. What is an Expression in Python?
Answer:
'' An Expression is a combination of values, variables, and operators.
It gives a result (output) when Python runs it.
Examples of expressions are math calculations or variable addition.
👉 Example:

x = 5 + 3 # Here, "5 + 3" is an expression and gives result 8

👉 Example:
y = 10 * 2 # Expression "10 * 2" gives 20

👉 Expression always returns a value.
🔹 2. What is a Statement in Python?
Answer:
A Statement is a complete instruction to the computer.
It does something, like create a variable or print a message.
A statement may not return any value but performs an action.
👉 Example:
x = 5 # This is an assignment statement (It stores 5 in x)
print(x) # This is a print statement (It shows the value of x on screen)
🔹 3. Difference Between Expression and Statement
Feature Expression Statement
Meaning Combines values and operators to give a value Gives an instruction to do something
Returns Always returns a result May or may not return a result
Example 5 + 3, x * 2 x = 10, print(x)
Purpose Calculate values Do actions in the program

👉 Example in One Program:

x = 5 + 3 # '5 + 3' is Expression (result: 8) and 'x = 5 + 3' is Statement
print(x) # Statement (shows 8 on screen)
🔹 4. Why are Expressions Important?

Expressions help us to calculate values easily.
They are used in assignments, conditions, and loops.
Without expressions, we cannot perform calculations.
👉 Example:
total = price * quantity # Calculates total cost
🔹 5. Why are Statements Important?
Statements give instructions to the computer.
They help organize the program flow step by step.
Without statements, the computer won’t know what to do.
👉 Example:

if x > 10:
print("x is big") # A statement for decision

✅ Short Simple Summary for Book
Expression → Calculates values → Returns result → Example: 5 + 3, x * 2
Statement → Gives instructions → May not return result → Example: x = 5, print(x)
Expressions are part of statements.
Both help Python programs work correctly.

3.Variables


🔹 1. What is a Variable in Python?

Answer:

A Variable is a name that stores data in computer memory.
It works like a box where we keep numbers or text.
We can use variables to store data and use it later in the program.
👉 Example:

python
x = 10
name = "Shubh"
print(x)
print(name)

Output:
10
Shubhbr

🔹 2. How to Create a Variable?

Answer:
We write the variable name and use = to give it a value.
The value can be a number or text.
👉 Example:

age = 20
message = "I am learning Python"
print(age)
print(message)
Output:

20
I am learning Python

🔹 3. Variable Types

Integer (int) → Whole numbers → Example:

python
x = 5
print(x)

Output:
5

Float (decimal numbers) → Example:
python
pi = 3.14
print(pi)
Output:

3.14

String (Text) → Example:
python
name = "Shubh"
print(name)

Output:

Shubh

Boolean (True / False) → Example:
python
flag = True
print(flag)

Output:

True

🔹 4. Rules to Create Variable Names

Start with a letter or underscore.

No spaces or special characters allowed.
👉 Example of correct variables:

python
my_name = "Shubh"
_user_age = 20
print(my_name)
print(_user_age)

Output:

Shubh
20

🔹 5. Changing Variable Values

Answer:
We can change variable values anytime.
👉 Example:

x = 5
print(x) # Output: 5

x = 10
print(x) # Output: 10

Output:

5
10

🔹 6. Why Variables Are Important?

Variables help store data to reuse later.
They make programs simple and easy to read.
👉 Example:

name = "Shubh"
print("Hello", name)

Output:
Hello Shubh

✅ Simple Summary for Book

Variables store data (numbers, text, True/False). Use = to assign value → Example: x = 5, name = "Shubh" Variable names should start with letter or underscore, no spaces. We can change variable values anytime. Output shows the stored data.




4.Avoiding Name Error when using Variables

🔹 1. What is Name Error in Python?

Answer:
Name Error happens when we use a variable that is not created yet.
It means Python does not know that variable.
This gives an error and stops the program.
👉 Example of Name Error:

print(x) # x is not defined yet → NameError


Output:

NameError: name 'x' is not defined

🔹 2. How to Avoid Name Error?

First, create the variable and give it a value.

Then use the variable later in the program.
👉 Correct Example:

x = 10
print(x)


Output:

10

🔹 3. Example – Wrong vs Correct Way

❌ Wrong Way (causes NameError):

python
print(age) # Error – age is not defined
age = 20

Output:

NameError: name 'age' is not defined

✅ Correct Way:

age = 20
print(age)

Output:

20

🔹 4. Best Practice to Avoid Name Error

Always define the variable before using it.

Avoid typos in variable names (case-sensitive).
👉 Example of Typo Error:

age = 20
print(Age) # NameError because 'Age' ≠ 'age'

Correct Example:

paython
age = 20
print(age) # Correct

✅ Simple Summary for Book

NameError happens if we use a variable before creating it.

Always assign a value first → Example: x = 10
Avoid typos → Example: age ≠ Age
Correct Example:
x = 5
print(x)
